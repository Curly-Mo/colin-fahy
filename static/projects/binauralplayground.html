Binaural Playground

Description:

An interactive interface for listening to 3-Dimensional audio

Features:

  • Uses the Matter.js physics engine for realistically updating the position of the sounds.
  • In addition to the built in Web Audio API Head-Related Transfer function, includes my own physical model for creating binaural audio by computing the Interaural Time Difference and Interaural Level Difference cues based on the objectâ€™s distance, azimuth and elevation angles. The theory behind the physical model and the original Python code can be found here.
  • Also includes a Web Audio API implementation of Crosstalk Cancellation for listening over stereo speakers. Though the limitations of the Web Audio API make it not work as well as my Python implementation.
