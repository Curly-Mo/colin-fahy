WebVR

Description:

A Virtual Reality interface for listening to 3-Dimensional audio. Inspired by the work I did with the original binaural playground, after I was dissapointed with the outcome of trying to represent 3D space in a 2D interface.

Features:

  • Runs on Oculus Rift, GoogleCardboard/GearVR as well as in any browser without the VR effect.
  • Upload the stems of a song to here each instrument as a separate moveable point source in 3D space.
