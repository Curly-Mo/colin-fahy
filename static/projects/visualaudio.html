VisualAudio

Description:

The first work I made when learning about the Web Audio API. See visualizations for the time domain and frequency domain content of the currently playing audio.

Features:

  • Play any text-to-speech, upload an audio file, or combine any number of sinusoidal oscillators.
  • View the oscilloscope, waveform, spectrum, spectrogram and the cepstrum of the currently playing sound.
  • Apply signal processing effects such as filtering, dynamic compression, and reverberation.
http://webaudio.cfahy.com/visualaudio
