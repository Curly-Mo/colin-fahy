Swingify

Description:

Makes any song swing by extending the first half of every beat and shortening the second half. Runs beat detection by estimating the tempo based on the correlation of an onset function, then picking the strongest onsets that are consistent with the estimated tempo. Uses a phase vocoder to timestretch the two halfs of each detected beat.
Deployed on an Amazon EC2 server with Flask and nginx to provide a web front-end for user's to access the Python script.
