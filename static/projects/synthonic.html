Synthonic

Description:

An audio additive synthesizer based on the concept of adjusting the ADSR (Attack Decay Sustain Release) envelope of each individual harmonic/partial, which should theoretically allow for the creation of any possible sound.

Features:

  • Visuals show what the current sound looks like and how it changes over time in both the time domain and frequency domain.
  • Works with any MIDI input device or the keyboard for playing notes.
  • Apply frequency modulation and amplitude modulation to each individual partial.
  • Add filtered noise (subtractive synthesis) for sounds that might not otherwise be possible without a very large number of sinusoids.
  • Presets show how this synthesizer could be used to mimic real instruments such as a bell or clarinet.
